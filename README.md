# 📘 Lista de Exercícios em Java

Este repositório contém uma coleção de **10 exercícios básicos em Java**, focados em lógica de programação, estruturas de repetição e validação de dados.

---

## 🚀 Objetivo

Praticar conceitos fundamentais da programação, como:

* Entrada e saída de dados (`Scanner`)
* Estruturas de repetição (`for`, `while`, `do-while`)
* Estruturas condicionais (`if`, `else`)
* Validação de dados
* Operadores lógicos e aritméticos

---

## 📂 Lista de Exercícios

### ✅ 1. Validação de Nota

Solicita uma nota entre 0 e 10.
Caso o valor seja inválido, o programa continua pedindo até receber um valor correto.

---

### ✅ 2. Usuário e Senha

Lê um nome de usuário e senha.
Não permite que a senha seja igual ao usuário.

---

### ✅ 3. Validação de Dados

Valida as seguintes informações:

* Nome: mais de 3 caracteres
* Idade: entre 0 e 150
* Salário: maior que 0
* Sexo: 'f' ou 'm'
* Estado civil: 's', 'c', 'v', 'd'

---

### ✅ 4. Crescimento Populacional (Fixo)

Calcula quantos anos são necessários para que a população A ultrapasse a população B.

* População A: 80.000 (crescimento 3%)
* População B: 200.000 (crescimento 1.5%)

---

### ✅ 5. Crescimento Populacional (Dinâmico)

Versão melhorada do exercício anterior:

* O usuário informa as populações e taxas
* Os valores são validados
* Permite repetir o cálculo

---

### ✅ 6. Números de 1 a 20

* Exibe números de 1 a 20 em linhas separadas
* Depois exibe na mesma linha

---

### ✅ 7. Maior Número

Lê 5 números e informa o maior.

---

### ✅ 8. Soma e Média

Lê 5 números e calcula:

* Soma
* Média

---

### ✅ 9. Números Ímpares

Exibe todos os números ímpares entre 1 e 50.

---

### ✅ 10. Intervalo entre Números

Recebe dois números e exibe todos os valores entre eles.

---

## 🛠️ Tecnologias Utilizadas

* Java (JDK 8 ou superior)
* IDE recomendada: IntelliJ IDEA, Eclipse ou VS Code

---

## ▶️ Como Executar

1. Instale o Java (JDK)
2. Compile o arquivo:

   ```bash
   javac NomeDoArquivo.java
   ```
3. Execute:

   ```bash
   java NomeDoArquivo
   ```

---

## 📚 Conceitos Trabalhados

* Laços de repetição
* Validação de entrada
* Operadores lógicos (`&&`, `||`)
* Manipulação de Strings (`equals`, `length`)
* Tipos primitivos (`int`, `double`, `char`)

---

## 💡 Observações

* Sempre utilize `.equals()` para comparar Strings
* Use `charAt(0)` para ler um único caractere
* Evite usar `==` com Strings

---

## 🎯 Conclusão

Essa lista é essencial para desenvolver uma base sólida em lógica de programação com Java.
Dominar esses exercícios facilita o aprendizado de estruturas mais avançadas no futuro.
